# git-learning

Репозиторий для изучения Git и управления версиями.

## Описание
Этот репозиторий создан для обучения работе с системой контроля версий Git.

## Команды Git
- `git status` - проверка статуса репозитория
- `git add` - добавление файлов в индекс
- `git commit` - создание коммита
- `git push` - отправка изменений в удаленный репозиторий
